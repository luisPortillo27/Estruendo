Campamento Estruendo 3.0
Aplicación web interactiva diseñada como herramienta digital de apoyo para actividades de campamento. Este proyecto ofrece recursos educativos, materiales descargables y una experiencia dinámica para los participantes.
Descripción
Campamento Estruendo 3.0 es una página web desarrollada en HTML, CSS y JavaScript que simula un entorno interactivo para exploradores del rey. Incluye acceso a materiales, tutoriales, retos espirituales y actividades organizadas mediante un sistema de acordeón intuitivo.
El sistema también integra funcionalidades dinámicas como:
o	Reloj en tiempo real 
o	Modal de bienvenida personalizado 
o	Descarga de recursos en PDF 
o	Enlaces a tutoriales externos 
o	Herramientas externas (brújula online) 
o	Sistema de desbloqueo de tareas por tiempo 
Características principales
o	Diseño temático inspirado en campamentos (colores, iconos y estilo visual) 
o	Responsive Design adaptable a dispositivos móviles 
o	Acordeón interactivo para organizar contenido 
o	Control de acceso por tiempo (activación/desactivación de tareas) 
o	Personalización del usuario mediante ingreso de nombre 
o	Gestión de recursos (PDFs, enlaces, videos) 
Tecnologías utilizadas
o	HTML5 
o	CSS3 (Variables y diseño moderno) 
o	JavaScript (Vanilla JS) 
Funcionalidades destacadas
Modal de bienvenida
Solicita el nombre del usuario antes de ingresar al contenido.
Reloj dinámico
Muestra fecha y hora actual en tiempo real.
Sistema de acordeón
Organiza los temas:
o	Fogata 
o	Nudos y amarres 
o	Brújula 
o	Construcciones 
o	Reto espiritual 
o	Tareas 
Sistema de tareas temporizado
o	Bloquea el acceso antes de una fecha 
o	Habilita contenido en un rango de tiempo 
o	Muestra cuenta regresiva
